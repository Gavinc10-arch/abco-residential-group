# ABCO Residential Group Website

Static launch site for Vercel.

## Before going live
1. Replace Google Analytics placeholder `G-XXXXXXXXXX` in all HTML files.
2. Replace Facebook Pixel placeholder `000000000000000` in all HTML files.
3. Add final phone number.
4. Confirm `info@abcoresidential.com` is active.
5. Upload all files to GitHub repository root and deploy on Vercel.
